# Insurance EDA Project

## Overview
Performed exploratory data analysis on an insurance dataset to understand factors affecting medical charges.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scipy

## Analysis Performed
- Data cleaning
- Missing value handling
- Outlier analysis
- Correlation analysis
- Pearson correlation
- Chi-square test
- Feature engineering
- Feature selection

## Important Findings
- Smoking has the strongest correlation with insurance charges.
- Age and BMI also influence charges.
- Certain categorical features have weak relationships.
